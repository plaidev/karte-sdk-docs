UserSync

@objc(KRTUserSync)
public class UserSync : NSObject, Codable

WebView 連携するためのクラスです。

Webページを開くWebViewに連携用のスクリプトを設定することで、WebとAppのユーザーの紐付けが行われます。
なお連携を行うためにはWebページに、KARTEのタグが埋め込まれている必要があります。

    指定されたURL文字列にWebView連携用のクエリパラメータを付与します。
