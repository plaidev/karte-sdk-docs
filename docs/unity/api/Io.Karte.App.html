RenewVisitorId()

ビジターIDの再生成処理を行います。

なお内部では、以下の処理が行われます。

  • プッシュ通知の配信許可フラグ (plugin_native_app_identity.subscribe) を 非許可 (false) に変更
  • 端末に保存されている設定値の削除
  • visitor_id の再発行
  • 新たに生成された visitor に対してFCMトークンを紐付け
