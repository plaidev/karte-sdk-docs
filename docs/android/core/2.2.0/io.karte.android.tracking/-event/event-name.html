eventName

val eventName: EventName

イベント名

イベント名には半角小文字英字 ( a-z )、半角数字 ( 0-9 ) と下線 ( _ ) のみ使用できます。 なお下線 ( _ ) で始まるイベント名はシステム上のイベントと干渉するため使用できません。
