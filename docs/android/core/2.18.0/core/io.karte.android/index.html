class KarteApp : ActivityLifecycleCallback

KARTE SDKのエントリポイントであると共に、SDKの構成および依存ライブラリ等の管理を行うクラスです。SDKを利用するには、KarteApp.setupを呼び出し初期化を行う必要があります。初期化が行われていない状態では、イベントのトラッキングを始め、SDKの機能が利用できません。なおアプリ内メッセージ等のサブモジュールについても同様です。SDKの設定については、初期化時に一部変更することが可能です。 設定を変更して初期化を行う場合は、Configを指定してKarteApp.setupを呼び出してください。
