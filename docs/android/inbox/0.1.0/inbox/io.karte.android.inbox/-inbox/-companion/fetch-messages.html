fetchMessages

suspend fun fetchMessages(limit: Int? = null, latestMessageId: String? = null): List<InboxMessage>?
Content copied to clipboard

Push通知の送信履歴を取得するSuspend関数です。エラー発生時はnullを返します。

Return

引数で指定した条件に合致するPush通知送信履歴のリストを返します。エラー発生時はnullを返します。

Parameters

limit

最大取得件数を指定します。デフォルトは最新50件を取得します。

latestMessageId

この値で指定されたmessageIdより前の履歴を取得します。指定したmessageIdを持つ履歴は戻り値に含まれません。
