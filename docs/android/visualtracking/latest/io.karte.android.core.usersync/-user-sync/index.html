UserSync

object UserSync

WebView 連携するためのクラスです。

WebページURLに連携用のクエリパラメータを付与した状態で、URLをWebViewで開くことでWebとAppのユーザーの紐付けが行われます。

なお連携を行うためにはWebページに、KARTEのタグが埋め込まれている必要があります。
