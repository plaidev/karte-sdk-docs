フィールド名の制限

  • ドット ( . ) を含む名称は使用できません。
  • $ で始まる名前は使用できません。
  • フィールド名として _source any avg cache count count_sets date f_t first keys l_t last lrus max min o prev sets size span sum type v を指定することはできません。

  • java.util.Date
  • Array
